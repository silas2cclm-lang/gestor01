V-Think 0.55 - Gestão de Projetos Premium

Como rodar:
1. Extraia o conteúdo deste ZIP para uma pasta.
2. Abra o arquivo 'index.html' em qualquer navegador (Chrome, Edge, Firefox).

Configuração de Segurança (Importante):
Para que as regras de acesso (Admin, Editor, Viewer) e a trava de segurança funcionem corretamente no servidor, você deve aplicar o conteúdo do arquivo 'firebase-rules.json' no seu console do Firebase:
- Acesse seu Console Firebase -> Realtime Database -> Rules.
- Cole o conteúdo de 'firebase-rules.json' e clique em Publicar.

Conteúdo:
- app.js (Lógica e Integração)
- index.html (Estrutura UI)
- style.css (Design Minimalista Premium)
- Logo/ (Imagens da marca)
- firebase-rules.json (Regras de Servidor)
